# 🎴 CHL: Kterou vrstvu dokumentace použít?

**Rychlá rozhodovací kartička** — kam napsat / kde hledat informaci.

---

## 🧠 BIG IMAGE — když otázka je „proč / co"

| Typický dotaz čtenáře | → Big Image |
|-----------------------|-------------|
| „Proč jsme to udělali takhle?" | ✅ |
| „Co to vlastně je / jak to funguje obecně?" | ✅ |
| „Jak spolu souvisí X a Y?" | ✅ |
| „Jaká jsou principy / pravidla systému?" | ✅ |

**Forma:** souvislý text + diagramy, pages + subpages
**Místo:** Mozkotron — sekce Referenční dokumentace

---

## 📋 SOP — když otázka je „jak"

| Typický dotaz čtenáře | → SOP |
|-----------------------|-------|
| „Jak konkrétně to mám teď udělat?" | ✅ |
| „Kterou ikonku mám kliknout?" | ✅ |
| „V jakém pořadí se dělají kroky?" | ✅ |
| „Co kontaktovat, když něco nejde?" | ✅ |

**Forma:** číslované kroky, max 1 obrazovka
**Místo:** Mozkotron — SOP knihovna (dle AOR skupin)
**Pojmenování:** `SOP_AORx_Sloveso-co`

---

## 🎴 CHL — když otázka je „rychle co"

| Typický dotaz čtenáře | → CHL |
|-----------------------|-------|
| „Kam to mám uložit? Rychle." | ✅ |
| „Která barva / teplota znamená co?" | ✅ |
| „Jaká jsou nejdůležitější pravidla?" | ✅ |
| „První týden jsem tu nový, na co si dát pozor?" | ✅ |

**Forma:** 1 A4 — tabulka, strom, diagram nebo krátký seznam
**Místo:** Mozkotron + vytištěné u stanic
**Pojmenování:** `CHL_nazev-tematu`

---

## ⚠️ Pravidla — nikdy nezapomeň

:::warning
1. **Big Image zatvrdne první.** Bez něj se SOP i CHL budou přepisovat.
2. **SOP odkazuje na Big Image.** Čtenář si může proklikat „proč".
3. **CHL se tvoří poslední.** Když se ukáže, co lidi opravdu potřebují rychle.
4. **Směr aktualizace je shora dolů.** Big Image → SOP → CHL. Nikdy ne obráceně.
5. **CHL není zdroj pravdy.** Zdroj pravdy je Big Image.
:::
