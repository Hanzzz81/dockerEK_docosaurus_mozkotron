## Zamítnuté přístupy pro page transition
1. CRT exit animace — stránka se unloadne před animací
2. CRT curtain overlay — zakrývá sidebar
3. CSS translateY — flash, ne pohyb
4. `scrollTo({behavior:'smooth'})` — prohlížeč příliš rychlý
5. Spacer-based scroll — nerozpoznatelné od žádného efektu
6. Jump+smooth scroll — UP směr nefunguje (negativní scroll neexistuje)

## Klíčové poučení
- Page transition (CRT) a TOC scroll jsou DVA oddělené systémy
- Pro řízený scroll vždy rAF smyčka, nikdy browser smooth
- `transform-origin: center 120px` pro vizuálně centrovaný scaleY
- Search panel: wrapper+inner+sibling handle (přežije innerHTML wipe)
- Dynamické panely: číst offsetWidth, ne konstanty
- Show flex container jako `display:'flex'`, ne `''`
