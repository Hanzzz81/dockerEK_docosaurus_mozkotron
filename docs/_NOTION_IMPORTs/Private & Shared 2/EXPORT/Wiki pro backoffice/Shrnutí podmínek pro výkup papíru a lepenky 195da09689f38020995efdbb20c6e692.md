# Shrnutí podmínek pro výkup papíru a lepenky

### **1️⃣ Nejlepší způsob třídění odpadu (doporučení)**

- **Lisované balíky** – nejlepší cena, hlavně pro vlnitou lepenku.
- **Třídění na druhy** – výhodnější než směsný sběr (noviny, časopisy, kancelářský papír).
- **Volný sběr (směsné papíry)** – méně výhodné, ale možnost pro menší množství.
- **Neodkupují**: mokré, laminované, termo papíry, dutinky.

### **2️⃣ Výkupní ceny (orientačně)**

| **Druh odpadu** | **Cena (Kč/kg)** |  |
| --- | --- | --- |
| **Kancelářský papír** | 3,50 |  |
| **Bílá vlnitá lepenka** | 5,50 (min. 20 kg) |  |
| **Směsný papír** | 1,50 – 2,00 (podle množství) |  |
| **Noviny/Letáky** | 2,80 – 3,30 (podle množství) |  |
| **Časopisy** | 2,80 – 3,30 (podle množství) |  |
- **Vyšší cena** za kvalitní třídění – nutná konzultace.
- **Lisované balíky** – individuální ceny dle podmínek MONDI.

### **3️⃣ Doporučení pro firmu (náš odpad)**

| **Odpad** | **Kam zařadit?** | **Cena (Kč/kg)** |
| --- | --- | --- |
| **Kancelářský papír (bílý)** | Kancelářský papír | 3,50 |
| **Kancelářský papír (barevný, samopropis, termo)** | Směsný papír | 1,50 – 2,00 |
| **Krabice od výrobků** | Směsný papír (nebo lisování) | 1,50 – 2,00 |
| **Letáky** | Letáky | 2,80 – 3,30 |
| **Plastové krabičky a fólie** | Nelze vykoupit | - |
