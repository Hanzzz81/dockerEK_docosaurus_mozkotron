# Třídění odpadů

Odpady ve firmě třídíme. Neděláme to jen kvůli posedlosti ekologií, ale abychom **ušetřili peníze**. Za dobře vytříděné odpady buď můžeme inkasovat nějaké peníze navíc do rozpočtu, nebo nebudeme platit tolik za likvidaci.

# Za co platíme a za co platí nám ?

### Materiály - za něco platíme a někdo může platit nám

| **platíme my** | lepenka a kartony | komunál |  |  |  |
| --- | --- | --- | --- | --- | --- |
| **zdarma** | PET | plasty |  |  |  |
| **zaplatí nám** | vytříděný papír | AL plechovky | Měď | železo | igelity a stretch |

### Spotřebiče - nic nedostaneme, nebo někdo může platit nám

| **nic nedostaneme** | černá | baterie |  |  |
| --- | --- | --- | --- | --- |
| **zaplatí nám VODICA** | praní a sušení |  |  |  |
| **zaplatí nám ELECTROWIN** | malá bílá bagy | chlazení | praní a sušení s X | sporáky |

# Jak likvidujeme - postupy

## Postup elektrospotřebiče VODICA

- dopředu hlásí, že přijede, obvykle to jsou pátky
- odebírá si pouze pračky, sušičky a myčky
- sám si vybírá, co si odebere zaplatí nám keš 400,- Kč za kus. Peníze se nosí do kanceláře
- co po něm zbyde a nechce to, označí se křížkem "X" aby bylo jasno co mu prošlo rukama a dá se elektrowinu

## Postup elektrospotřebiče - ELEKTROWIN

- [@Michal Pažický](https://javascript: void;)﻿ objednává předem přes B2B elektrowinu Recos v pořru pro
    - VB chlazení
    - ostatní VB
    - bagy
- jakmile dorazí, mají naložit množství z objednávky
- pokud toho máme víc, snažíme se na místě domluvit aby toho odebrali víc
- chlazení nakládáme vše, z ostatního to, co je označeno křížkem
- podepíšeme papíry co jsme předali a odnáší se do kanceláře
