# pro řidiče

Před blokem rozvozů se připojím do aplikace a podívám se co mě vše čeká za rozvozy.

Postupně naložím auto spotřebiči, ideálně od nejposlednějšího rozvozu po první.

Po montáži u zákazníka, vyfotím výrobek z více stran + vyfotím štítek se seriovým číslem.

Nechám si v aplikaci podepsat převzetí.
