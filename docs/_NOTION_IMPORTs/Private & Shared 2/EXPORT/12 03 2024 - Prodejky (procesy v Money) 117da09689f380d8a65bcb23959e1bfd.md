# 12.03.2024 - Prodejky (procesy v Money)

## **Prodejka vydaná - import a zaúčtování**

Prodejky, které běžně vznikají v PSQL se v určitých intervalech přenášejí do Money.

Nově se nebudou přenášet po 17:30 automaticky, ale je potřeba jejich import **vyvolat ručně**.

Tuto akci vyvoláme následujícím způsobem:

**Prodejní místa → Centrála PSQL → Prodejní místa Centrály PSQL**

![](https://t4612515.p.clickup-attachments.com/t4612515/0d7df956-971f-482f-b705-1159f7bc82bc/image.png)

**Vybereme konkrétní kasu → pravé tlačítko myši → import → dávkový import**

![](https://t4612515.p.clickup-attachments.com/t4612515/8e26a08f-491f-4e1f-a450-81fe66db92f8/image.png)

## **Hromadné zaúčtování prodejek**

Při zaúčtování prodejek, které jsou hrazené hotově vzniká pokladní doklad. Abychom zamezili zbytečnému vytváření více pokladních dokladů, budeme provádět zaúčtování jak klasických prodejek, tak zakázkových prodejek **současně**.

Abychom viděli všechny prodejky v dané pokladně včetně zakázkové řady, po vybrání pokladny je nutné zapnout funkci: **včetně podřízených**.
