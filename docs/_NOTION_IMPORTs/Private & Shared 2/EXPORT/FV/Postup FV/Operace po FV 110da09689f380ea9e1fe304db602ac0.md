# Operace po FV

Po vytvoření FV se obvykle:

- tiskne doklad - pokud je zákazník na místě
- posílá doklad mailem - pokud je zákazník vzdálený
- hradí faktura - pokud je faktura placená obratem hotovostí nebo kartou

## FV do mailu:

Pokud je na FV uložen partner s kontaktem „**email-fakturační**“, automat automaticky FV odešle na jeho email. Pokud není na firmě uložený fakturační mail, vyhodí se error, ale dá se adresa doplnit ručně. Pokud mail poslat nechcete, dá se tento dialog zrušit.
